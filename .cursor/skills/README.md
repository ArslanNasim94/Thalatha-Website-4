# THALATHA — CURSOR SKILLS DIRECTORY

This folder contains modular skill files that Cursor reads when implementing specific parts of the project.
Each skill file is self-contained and describes exactly what to build and how.

## Skill Files

| File | What it covers |
|---|---|
| `skill-setup.md` | Initial setup, installs, App.jsx wiring |
| `skill-cursor-ui.md` | Custom cursor, scroll progress bar, preloader |
| `skill-hero.md` | Hero section (3D scene, text reveal, CTAs) |
| `skill-stats.md` | Stats bar with animated counters |
| `skill-howitworks.md` | Scroll-pinned 3-step narrative section |
| `skill-features.md` | Glassmorphism feature cards grid |
| `skill-services.md` | Rotating service wheel + mobile carousel |
| `skill-apppreview.md` | 3D phone mockup showcase |
| `skill-testimonials.md` | Coverflow slider |
| `skill-download.md` | Gradient CTA section with particle trail |
| `skill-navbar.md` | Sticky frosted-glass navbar |
| `skill-footer.md` | Clean minimal footer |
| `skill-effects.md` | Global wow effects (morphing SVG, orbs, number scramble) |
| `skill-animation-utils.md` | Reusable GSAP patterns for the whole project |
